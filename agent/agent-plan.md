# Agent Usage Plan Definition

## Strategy

1. Assign Each Agent with detail context and let them collaborate.

2. Describe Task Content as detail as possible.

3. Guide Agent follow Constitution in the package.

4. Let individual agent do their job based on Constitution.

5. Give each agent specific role, and make them plan and work.

6. Let them evaluate their output if possible.

7. Human in the Loop while aggregating.
